AWS Backup

  • AWS Backup is a simple and cost-effective way to protect your data by backing up your Amazon EFS file systems. Amazon EFS is natively integrated with AWS Backup. You can use the EFS console, API, and AWS Command Line Interface (AWS CLI) to enable automatic backups for your file system.

Application Manager reduces the time taken by the DevOps engineers to detect and investigate issues with AWS resources.
