Amazon EKS Anywhere is a deployment option for Amazon EKS that allows customers to create and operate Kubernetes clusters on customer-managed infrastructure, supported by AWS.

  • Amazon EKS Distro includes open-source Kubernetes components, Kubernetes Control Plane components (kube-controller-manager, etcd, and CoreDNS), Kubernetes Node components (kubelet, Kubernetes CSI, and CNI), and command-line clients (kubectl and etcdctl).
