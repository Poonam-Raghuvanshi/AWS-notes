Amazon Redshift

Fully-managed petabyte-scale cloud-based data warehouse product designed for large scale data set storage and analysis.

Amazon Redshift Spectrum

Efficiently query and retrieve structured and semistructured data from files in Amazon S3 without having to load the data into Amazon Redshift tables and supports nested data types. It resides on dedicated Amazon Redshift servers that are independent of your cluster and queries use much less of your cluster's processing capacity.

Advanced Query Accelerator or AQUA for Redshift

Query performance by 10X. It resolves network bandwidth and memory processing (CPU) bottleneck, low cost, and is easy to deploy.
